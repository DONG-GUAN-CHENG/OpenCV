### 影像處理簡單介紹

 *cv2.namedWindow(視窗名稱,視窗旗標參數)
 flag如下:
 1.
WINDOW_NORMAL:可自行調整視窗大小
WINDOW_AUTOSIZE:按照影像調整視窗，user無法調整,(default)
